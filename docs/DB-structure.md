  핵심 트랜잭션: NFC 검증 → NFT 발행 → 리워드 발급

  아래 3단계가 단일 트랜잭션 안에서 원자적으로 처리되어야 함:

  1. nfc_tags WHERE tag_uid = ? → step_id 조회
  2. 검증:
     - event_participations 존재 여부 (해당 사용자 참여 확인)
     - step_completions 미존재 여부 (중복 완료 방지)
     - 이전 step 완료 여부 (순서 강제)
     - events.status = active or featured (종료/예정 이벤트 차단)
  3. step_completions INSERT
