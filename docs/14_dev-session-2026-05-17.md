### SectionHeader
- `description` prop 추가 — 섹션 제목 아래 소제목 표시 가능

### 오픈예정 캠페인 (UpcomingEventCard)

- 변경: 제목 → 지역 → (D-N일 후 오픈 + 리워드) 순서 — 웹 레이아웃 일치
- `rewardLabel` 필드 추가 (`adaptEventSummary`)

### 국기 이모지
- `adapters.js` `COUNTRY_FLAGS`에 `'대한민국': '🇰🇷'` 추가
- 매핑 없는 국가 fallback `'🌏'` → `null` 변경
- `ActiveEventCard`, `UpcomingEventCard`에서 국기 표시 제거 (작은 카드는 불필요)
