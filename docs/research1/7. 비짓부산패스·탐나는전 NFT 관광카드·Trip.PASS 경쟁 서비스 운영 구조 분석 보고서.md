# 비짓부산패스·탐나는전 NFT 관광카드·Trip.PASS 경쟁 서비스 운영 구조 분석 보고서
**Land-In 관점의 국내 유사 서비스 법·제도·운영 구조 비교 조사**
*작성 기준일: 2026년 5월 12일*

***

## 1. Executive Summary

세 서비스는 각각 뚜렷이 다른 DNA를 가지고 있다.

- **비짓부산패스(Visit Busan Pass)**: 지자체(부산시·부산관광공사) 주도의 **관광지 무료입장권 묶음 + 교통카드 결합형** 패스. 전자금융이 아닌 관광상품에 가까우며, 외국인 전용이지만 '결제 수단'이 아닌 '이용권' 구조다.[^1]
- **탐나는전 NFT 관광카드(나우다 포함)**: 제주도 지역화폐 '탐나는전'을 기반으로 한 **지역화폐 + 멤버십/NFT 디지털 관광증** 이중 구조. 법적으로는 '지역사랑상품권(지역화폐)'이며, 외국인등록증 없는 단기 관광객은 탐나는전 본체에서는 원칙적으로 배제된다.[^2]
- **Trip.PASS(트립패스)**: 로드시스템(플랫폼·eKYC)과 신한카드(금융인프라)가 협력한 **여권 기반 선불전자지급수단 + 교통카드 + 신원인증 통합 플랫폼**. 단기 체류 외국인을 주 타겟으로 하며, 전자금융거래법상 선불전자지급수단에 가장 근접한 서비스다.[^3][^4]

**Land-In과의 핵심 접점:** Trip.PASS는 여권 eKYC와 선불결제라는 점에서 Land-In과 가장 직접적으로 경합하며, 비짓부산패스는 지역 관광 제휴 구조에서 참고 모델이 된다. 탐나는전은 지역화폐 직접 연동을 주장할 경우 충돌하는 제도 병목을 보여주는 반면, 나우다(NFT 디지털 관광증)는 Land-In의 NFT 보상 레이어와 협력 가능성이 있다.

## 2. 경쟁 서비스별 운영 구조 분석

### 2.1 비짓부산패스 (Visit Busan Pass)

**한 줄 정의:** 부산 주요 관광시설 무료입장 + 대중교통 이용이 가능한 외국인 전용 실물/모바일 관광패스.

#### 운영 주체
- **발주·기획:** 부산광역시(관광정책과)
- **실운영:** 부산관광공사 (공기업, 부산시 위탁)
- **운영 대행사:** 민간 대행사에 재위탁 — 2025년 기준 사업 운영 공고 약 5.2억 원 규모로 별도 민간 사업자 선정[^5]
- **교통카드 기능:** 로카모빌리티(Mybi 계열) 교통 인프라 연동[^6]
- **OTA 판매:** 클룩(Klook), 케이케이데이(KKday), 트립닷컴(Trip.com), 라쿠텐트래블, 와우패스 등[^7][^1]

#### 참여 기관 구조
부산시(정책·예산) → 부산관광공사(위탁 운영) → 민간 대행사(실물 운영·키오스크) → 로카모빌리티(교통카드 인프라) → OTA(유통·판매). 제휴 관광시설(무료 가맹점 39개소, 할인 가맹점 172개소 — 2025년 4월 기준 총 211개소)[^8]

#### 상품 구조
- **유형:** 관광지 자유이용권형 + 충전형 교통카드 결합
- **권종:** 시간제한형(24시간권 49,000원, 48시간권 69,000원) + 수량제한형(BIG3, BIG5)
- **형태:** 실물 카드 + 모바일 패스 (단, 모바일 패스는 관광지 전용, 대중교통 이용 불가)[^9]
- **혜택:** 무료 가맹점 무제한 입장, 할인 가맹점 특별 혜택, 시티투어버스 무료, 대중교통 충전 후 이용 가능
- **판매 실적:** 2024년 말 기준 누적 31만 2,826매 판매, 총 판매금액 217억 원. 2024년 한 해 22만 1,695매(전년 대비 약 2.5배 증가)[^10]

#### 외국인 UX 흐름
1. OTA(Klook, KKday 등) 또는 visitbusanpass.com에서 사전 구매
2. 부산 도착 후 관문지역(김해공항·부산항 등) 판매처에서 실물카드 수령 (또는 모바일 앱에서 QR로 즉시 사용)
3. 관광지에서 카드/QR 제시 → 무료 입장
4. 교통카드 사용 시 편의점 또는 지하철역에서 별도 현금 충전 필요
5. 비짓부산 앱 다운로드 → QR 스캔으로 패스 등록 → 만료일 확인

**마찰 요소:** 실물카드 수령을 위해 관문지역 방문 필수, 대중교통 이용을 위한 별도 현금 충전 필요, iOS 모바일 교통카드 기능 제한, 외국 신용카드(Visa/Mastercard)로 대중교통 직접 이용 불가[^9]

#### 결제·정산 구조
- 패스 자체는 **선불 관광이용권(쿠폰·바우처 성격)**으로 전자금융거래법상 선불전자지급수단에 해당하지 않는다.
- 관광시설 무료 입장 비용: 부산관광공사가 가맹점과 사전 협약 → 패스 판매 수익에서 가맹점에 정산(이용횟수 기준 또는 정액 방식)
- 부산시 예산 보조금과 패스 판매 수익이 주 재원[^11]
- 부산관광공사 3년 연속 흑자(2026년 4월 기준) 달성, 수익 모델로 자리잡은 것으로 평가[^12]

#### 기술 구조
- 앱 기반 QR 관리 시스템 (비짓부산패스 공식 앱)
- 교통카드: 로카모빌리티(Mybi) 캐시비 인프라 연동
- 블록체인/NFT: 없음
- 언어 지원: 영어·일어·중국어(간체·번체) 4개 언어[^13]

#### 법·제도 구조
- **법적 성격:** 선불 관광이용권(쿠폰·바우처). 전자금융거래법상 선불전자지급수단 **아님**
- **외국인 가입:** 여권 없이도 구매 가능, 외국인등록증 불필요, 한국 통신사 인증 불필요
- **충전 방식:** 패스 구매는 해외카드 OTA 결제로 가능. 교통카드 충전은 국내 편의점·지하철역 현금 충전
- **가맹점 수수료:** 공개 자료 없음. 부산관광공사와 가맹점 간 협약 기반 추정

#### 수익·예산 구조
- 재원: 부산시 보조금 + 패스 판매 수익 + OTA 협찬금
- 2024년 민간 BOF 행사 기준 부산시 보조금 20억 원, 민간사업자 대행사업비 20억 원, 부산관광공사 자체 사용 예산 6억 원 규모[^11]
- 패스 판매가의 일부는 무료가맹점 이용 비용 정산에 사용되므로, 순수익 마진은 제한적

#### 한계
- 외국인의 '결제 수단'이 아닌 '이용권 묶음'이므로, 범용 결제 기능 없음
- 대중교통 외국인 직불 결제 환경 미비 (비자·마스터카드 교통 단말기 2027년 도입 목표)[^9]
- 부산 단일 지역 서비스, 다지역 확장 구조 없음
- 모바일 패스의 교통카드 기능 제한(특히 iOS)

#### Land-In 관점 시사점
- 비짓부산패스는 '관광패스 + 가맹점 제휴' 모델의 성공 사례로, Land-In의 지역 제휴 구조 설계 시 벤치마크 가능
- 그러나 단일 지역·단일 이용권 구조이며 외국인 결제 인프라가 취약하여 Land-In이 주장하는 '다자 정산 + 선불결제 + 관광 보상' 구조와 직접 경쟁하지는 않음
- 비짓부산패스 가맹점 네트워크(211개소)를 Land-In의 부산 지역 온보딩 파트너로 활용할 가능성 있음

### 2.2 탐나는전 NFT 관광카드 (나우다 포함)

**한 줄 정의:** 제주특별자치도 발행 지역화폐 '탐나는전'과 NFT 기반 디지털 관광증 '나우다(NOWDA)'의 이중 구조로 운영되는 제주 특화 관광·결제 플랫폼.

#### 탐나는전 vs. 나우다 구분
이 두 서비스는 별도의 사업체이면서도 긴밀히 연계된다.
- **탐나는전:** 제주도민 대상 지역화폐(선불전자지급수단). 관광객도 사용 가능하나 외국인등록증 보유 외국인만 정식 가입 가능[^2]
- **나우다(NOWDA):** 제주관광공사 발행 NFT 기반 디지털 관광증. 내국인 관광객 대상(만 14세 이상). 혜택 수령 시 탐나는전 지역화폐 지급[^14][^15]

#### 운영 주체
- **탐나는전:** 제주특별자치도(발행·감독) + 운영대행사 컨소시엄
  - 2023~2025: 제주은행–나이스정보통신 컨소시엄[^16]
  - 2026년부터: 제주은행–비즈플레이 컨소시엄 (2025년 11월 선정, 3년 계약)[^17]
  - 운영 수수료: 발행액의 0.1% (이전 0.9%에서 대폭 인하)[^18]
- **나우다:** 제주관광공사 주도, 네이버파이낸셜과 MOU(2025년 6월)[^19]
  - 네이버페이 월렛 연동, NFT 기반 비수탁형 디지털 지갑 구조

#### 참여 기관 구조
제주도(정책·발행) → 운영대행사(비즈플레이+제주은행 컨소시엄) → 가맹점(QR 수수료 0원) → 사용자
나우다: 제주관광공사 → 네이버파이낸셜(월렛 연동) → 제휴 관광사업체(160개소 이상) → 사용자[^14]

#### 상품 구조

**탐나는전:**
- 유형: 지역화폐(선불전자지급수단 성격)
- 종류: 종이상품권·카드형·모바일형(QR/바코드) 3종
- 혜택: 10% 캐시백(월 최대 7만 원), 소득공제(전통시장 40%, 일반 30%), 제주도 내 소상공인 가맹점 사용
- 충전한도: 1회 50만 원, 월 100만 원, 충전일로부터 5년 유효[^20]
- 사용불가: 백화점, 대형마트, 대기업 직영 프랜차이즈, 특급호텔 등

**나우다(디지털 관광증):**
- NFT 기반 디지털 관광증. 발급 시 '제주와의 약속' 서약 필요[^15]
- 혜택: 제휴 사업체 10% 이상 할인(현재 160개소 이상)[^14]
- 멤버십 등급: 웰컴(기본) → 스페셜(25회) → 시그니처(50회)
- 탐나는전 지역화폐 직접 지급: 단체 관광객 1인당 3만 원, 개별 관광객 최대 5만 원(추첨)[^19]
- 출시: 2025년 8월 11일 시범 오픈, 9월 26일 정식 오픈
- 2025년 10월 기준 가입자 4만 500명 돌파 (출시 약 2개월 만)[^14]

#### 외국인 UX 흐름 — 탐나는전 본체
1. 외국인등록증 보유 외국인만 앱·은행 방문 가입 가능
2. 한국 통신사 본인인증 필요 (SMS 기반)
3. 국내 은행계좌 연결 → 충전
4. 제주도 내 가맹점 QR/바코드 결제
5. **단기 체류 외국인 관광객(외국인등록증 미보유)은 사실상 사용 불가**[^2]

#### 외국인 UX 흐름 — 나우다
- 네이버 계정 또는 네이버페이 계정 보유자만 가입 가능 (국내 서비스 의존)
- 현재 내국인 만 14세 이상만 정식 대상
- 2026년 외국인 여행 플랫폼 확장 계획 언급, 아직 미시행[^21]

#### 결제·정산 구조

**탐나는전:**
- QR 결제 수수료 0원 (가맹점 부담 없음) — 운영비는 발행액의 0.1% 수수료와 제주도 예산으로 충당[^22]
- 인센티브(10% 캐시백) 재원은 제주도 정책예산
- 제로페이 기반으로 알리페이, 라인페이, 페이티엠 등 18개국 37개 앱 연동 (외국인 QR 결제 인프라 병행)[^23]

**나우다:**
- 제휴 사업체가 자체 10% 이상 할인 제공 (제주관광공사가 비용 일부 보전 추정, 공개 자료 없음)
- 지역화폐 지급분은 관광진흥 예산 사용

#### 기술 구조

**탐나는전:**
- 비즈플레이 플랫폼 기반 앱/카드/QR 운영 (2026년 1월 9일 신규 플랫폼 전환)[^24]
- 블록체인 없음 (중앙화 DB 기반 지역화폐 정산)
- 제로페이 QR 인프라 연동

**나우다:**
- NFT 기반 비수탁형 디지털 지갑 (구체 블록체인 체인 공개 미확인)
- 네이버페이 월렛 연동 — 발급, QR 인증 일원화[^19]
- 사용자 선택 행동(교통수단, 숙소, 체험) 데이터 온체인 기록 구조
- NFT 발급 = 관광 인증서. 미션 완료 시 디지털아트 NFT 추가 제공[^15]

|------|---------|-------|
| 법적 성격 | 지역사랑상품권(지역화폐) — 전자금융거래법 적용 가능 | 멤버십 디지털 배지(관광증) — 전자금융거래법 외 |
| 외국인 사용 | 외국인등록증 필요 (단기 관광객 불가) | 현재 내국인만. 2026년 외국인 확장 예정 |
| 환금성 | 총 충전액 60% 이상 사용 시 잔액 환불 가능[^20] | 없음 (관광 혜택만) |
| NFT 규제 리스크 | 해당 없음 | 낮음 — 단순 디지털 배지·인증서 성격, 환금성·거래 기능 없음[^15] |
| 전국 확장 | 불가 — 지역화폐법상 제주도 내 사용 한정 | 이론상 가능하나 제주관광공사 주관 |

#### 수익·예산 구조
- 탐나는전 운영: 발행액 0.1% 수수료 + 제주도 예산 보조
- 인센티브 캐시백: 관광진흥 사업 예산, 중앙정부 지역화폐 지원금 일부 활용
- 나우다: 제주관광공사 관광진흥 예산 + 네이버파이낸셜 협업비

#### 한계
- **핵심 문제:** 단기 체류 외국인 관광객이 탐나는전 본체를 직접 사용할 수 없음 — "외국인 관광 화폐"라는 홍보와 실제 접근성 간 갭 존재
- 나우다는 아직 내국인 한정, 외국인 접근 불가
- 네이버 계정 의존 구조 → 외국인 관광객 진입장벽
- 제주도 단일 지역 한정, 구조적으로 전국 확장 불가
- 나우다 초기 이슈: 공영관광지 미포함, 기존 탐나오 플랫폼과 혜택 중복 지적[^21]
- 탐나는전 운영대행사 교체(2026년) 시 플랫폼 전환 기간 서비스 중단 발생[^25]

#### Land-In 관점 시사점
- 탐나는전 직접 연동을 주장하면, **외국인등록증 없는 관광객 배제 문제**와 **지역화폐법상 사용처 제한** 두 가지 제도 병목에 즉시 부딪힘
- 나우다와의 협력(NFT 관광 보상 레이어 공유)은 실현 가능성 있음 — Land-In의 관광 NFT 보상이 나우다 미션 완료 증빙으로 연계되거나, 탐나는전 캐시백을 Land-In 보상으로 전환하는 간접 연동 모델 검토 가능
- 제주관광공사·네이버파이낸셜과의 MOU 구조가 Land-In 외국인 온보딩에 참조 모델

### 2.3 Trip.PASS (트립패스)

**한 줄 정의:** 로드시스템의 여권 eKYC 기술과 신한카드의 금융 인프라를 결합한 외국인 전용 선불전자지급수단 + 교통카드 + 신원인증 통합 플랫폼.

#### 운영 주체
- **플랫폼·eKYC:** 로드시스템 (대표 장양호) — 모바일 여권 플랫폼 기업[^26]
- **금융 서비스:** 신한카드 (2025년 3월 공식 리브랜딩 출시)[^3]
- **초기 결제 인프라:** BC카드 (2024년 3월 카드 출시 시점) → 신한카드로 전환[^27]
- **협력 채널:** 이스타항공(기내 판매), 인터파크 트리플(사전 구매), CU/GS25 편의점(수령·충전)[^28]
- **신원인증:** 로드시스템 자체 여권 OCR + 안면인증 기반 eKYC

#### 참여 기관 구조
로드시스템(eKYC 플랫폼 구축·운영) + 신한카드(선불카드 발급·결제·마케팅) + 항공사/OTA(유통) + 편의점(충전·수령) + 이동통신/택시(모빌리티 서비스). 양사 역할 명확히 분리: 로드시스템이 여권 eKYC 담당, 신한카드가 금융 서비스·마케팅 담당[^4]

#### 상품 구조
- **유형:** 기명식 선불전자지급수단 (전자금융거래법 적용)
- **카드 종류:** 선불카드 2종 (모바일 가상카드 + 실물 플라스틱 카드)
- **충전 한도:** 해외에서 최대 50만 원, 국내 입국 후 eKYC 완료 시 최대 200만 원[^29][^26]
- **결제:** 신한카드 전국 가맹점 + 신한 모바일 바코드 결제
- **교통:** 지하철, 버스, 택시 전국 이용 가능[^29]
- **추가 기능:** 택스리펀드(부가세 즉시 환급), 모바일 신원확인(면세점), 회원 간 무료 송금, i.M택시 호출[^28]
- **다운로드:** Google Play 10만+ 다운로드[^30]

#### 외국인 UX 흐름
1. 트립패스 앱 다운로드 (iOS/Android)
2. 실물 전자여권 NFC 스캔 또는 사진 촬영 → QR 형태 모바일 여권 생성
3. 안면인증 기반 비대면 eKYC → 외국인등록증·한국 통신사 인증 불필요[^27]
4. 해외 신용카드(Mastercard, AMEX, JCB 등)로 앱 내 충전 (최대 50만 원)
5. 국내 입국 후 모바일 여권 기반 추가 신원확인 → 한도 200만 원 기명카드 전환
6. 인천공항 CU 편의점에서 실물카드 수령 가능[^28]
7. 편의점 원화 충전, 해외카드 충전 병행 지원
8. 신한카드 전국 가맹점, 교통카드, 택시, 면세점 등 사용

**마찰 요소:**
- 전자여권(ePassport) 보유 필요 (NFC 칩 없는 구형 여권은 제한)
- iOS 모바일 교통카드 기능 제한 → 실물카드 수령 필요 (인천공항 CU만 가능)
- 충전 한도 200만 원 상한 — 장기 체류 외국인에게는 부족할 수 있음
- 앱 기반 서비스로 스마트폰 필수

#### 결제·정산 구조
- **선불전자지급수단:** 신한카드가 전자금융거래법상 선불전자지급수단 발행 라이선스 보유
- 결제 처리: 신한카드 결제 네트워크 (전국 가맹점)
- 초기(2024년)에는 BC카드 343만 가맹점 인프라 사용 → 2025년 신한카드 리브랜딩 후 신한 인프라 전환[^31]
- 택스리펀드: GS25, CU 등 가맹점에서 부가세 즉시 환급[^26]
- 수익 구조: 충전 수수료(현재 무료 프로모션), 향후 수수료 수익·가맹점 수수료 기대. 빅데이터 사업(eKYC 활용) 추진[^4]

#### 기술 구조
- **eKYC:** 로드시스템 자체 여권 OCR + NFC + 안면인증 기반 비대면 신원확인
- **블록체인 DID:** 여권 진위 확인에 블록체인 DID(탈중앙화 신원확인) 활용[^26]
- **결제:** EMV 규격 QR결제 + NFC 교통카드 + 실물카드
- **개인정보:** 여권 정보·안면 데이터 처리 — 개인정보보호법상 민감정보 처리 및 동의 필수
- NFT/관광 보상: 현재 없음

#### 법·제도 구조
- **법적 성격:** **전자금융거래법상 선불전자지급수단** — 신한카드 라이선스 기반 발행
- **외국인 가입:** 여권만으로 가능. 외국인등록증·한국 통신사 인증·국내 은행계좌 불필요[^27]
- **KYC:** 전자여권 NFC + 안면인증 = eKYC. 외국인등록증 발급 전 금융 소외 계층도 이용 가능[^32]
- **충전:** 해외카드 직접 충전 가능 (Mastercard, AMEX, JCB 지원)
- **AML:** 신한카드가 AML(자금세탁방지) 의무 부담
- **탑승 데이터 연계:** 이스타항공 기내 판매 채널 연계 있으나, 실시간 탑승 데이터 연동 여부는 공개 자료 미확인 (마케팅 채널 활용 수준)[^26]
- **전국 확장:** 신한카드 가맹점 = 전국 커버. 법적으로 전국 서비스 가능

#### 수익·예산 구조
- 민간 사업 (공공예산 없음)
- 로드시스템 수익: eKYC 플랫폼 운영 수수료, 라이선스 B2B 확장
- 신한카드 수익: 결제 수수료, 카드 발급 수수료, 제휴 혜택 마케팅 비용
- 빅데이터 사업(eKYC 기반 외국인 소비 데이터) 확장 계획[^4]

#### 한계
- 관광 보상·NFT·지역화폐 연계 없음 → 단순 결제 플랫폼
- 지역별 관광 혜택 제휴 구조 미흡 (비짓부산패스처럼 무료입장 가맹점 없음)
- iOS 실물카드 수령이 인천공항 CU만 가능 — 김포·제주·부산 공항 확장 미확인
- 관광 NFT·보상 레이어가 없어, 반복 방문 유도 매커니즘 부재
- 신한카드 단독 의존 구조 — 경쟁 카드사 배제

#### Land-In 관점 시사점
- **가장 직접적인 경쟁자:** 여권 eKYC + 선불전자지급수단 구조가 Land-In의 외국인 온보딩·결제 핵심 레이어와 동일
- 신한카드·로드시스템이 이미 시장 선점 중이므로, Land-In이 독립 선불전자지급수단 발행 노선을 취할 경우 라이선스·인프라 비용에서 크게 불리
- **협력 후보로도 유효:** Trip.PASS를 결제 백본(L2 레이어)으로 활용하고, Land-In은 NFT 보상·관광 혜택·지역 제휴 레이어를 추가하는 분업 구조 가능
- Trip.PASS에 없는 것: 항공 데이터 연계, 관광 NFT 보상, 지역화폐 연동, 다자 정산 구조

| Land-In과의 협력 가능성 | 높음 (지역 가맹점 네트워크) | 중간 (나우다 NFT 레이어 협력) | 중간~높음 (결제 백본 활용) |

## 5. Land-In 관점의 전략 해석

### 가장 위협적인 경쟁자

**Trip.PASS가 가장 직접적 경쟁자다.** 여권 eKYC + 선불전자지급수단 + 교통 통합 구조가 Land-In의 외국인 결제 레이어와 정확히 겹친다. 신한카드라는 대형 금융기관이 이미 전국 가맹점 인프라를 제공하고 있어, Land-In이 독자 선불전자지급수단 발행 노선을 택할 경우 규모 경제와 브랜드에서 열위에 놓인다.[^3][^4]

### 경쟁자보다 제휴·백본 후보

**Trip.PASS는 경쟁자이면서 동시에 결제 백본 후보**이기도 하다. Land-In이 독자 선불카드 발행 대신 Trip.PASS를 결제 레이어(L2)로 활용하고, Land-In은 항공 탑승 데이터 기반 NFT 보상·관광 혜택·다자 정산 레이어(L3~L4)를 얹는 분업 구조가 현실적이다. 비짓부산패스의 가맹점 네트워크(부산 211개소)는 지역 온보딩 협력 파트너로 적합하다. 나우다(제주관광공사)는 NFT 보상 레이어에서 공동 설계 가능성이 있다.[^19]

### 세 서비스가 해결하지 못하는 공백

1. **항공 탑승 데이터 기반 입국 즉시 온보딩:** 세 서비스 모두 공항 도착 전·후 항공 데이터와 연계된 자동 보상 지급 구조가 없다.
2. **다지역 통합 관광 보상:** 부산·제주·서울 등 복수 지역을 여행하는 외국인에게 단일 플랫폼에서 관광 인증+보상+결제를 제공하는 서비스가 없다.
3. **외국인 단기 관광객의 지역화폐 간접 수혜:** 지역화폐 혜택이 외국인 단기 관광객에게는 구조적으로 닫혀 있다.
4. **NFT 기반 관광 이력 소유권:** 여행 기록을 사용자가 소유할 수 있는 NFT 구조가 없다(나우다는 존재하나 내국인·제주 한정).

### Land-In의 가장 설득력 있는 차별화 논리

"**항공 탑승 기록을 트리거로 외국인 관광객에게 입국 즉시 선불 결제 + NFT 관광 보상을 자동 지급하는 단일 플랫폼**"이 핵심 포지셔닝이다. Trip.PASS는 결제만, 비짓부산패스는 이용권만, 나우다는 내국인 관광증만 — 이 세 조각을 통합하는 구조가 Land-In의 공백 채우기 논리다.

### 기존 서비스가 Land-In보다 강한 부분

- **비짓부산패스:** 부산 로컬 가맹점 커버리지(211개소), 관광시설 무료입장 제휴 신뢰도, 3년 이상의 외국인 대상 운영 경험
- **탐나는전:** 제주도민·소상공인 생태계와의 깊은 연계, 정부 예산 기반 10% 캐시백이라는 파격 혜택 — 민간이 재현하기 어려운 수준
- **Trip.PASS:** 신한카드 전국 가맹점 인프라, 이미 상용화된 여권 eKYC, 택스리펀드 등 완성도 있는 금융 서비스

### 지역화폐 직접 연동 시 법·제도 병목

Land-In이 직접 지역화폐 연동을 주장할 경우 **가장 먼저 막히는 지점은 두 곳**이다.
1. **행정 병목:** 지역사랑상품권 이용 활성화에 관한 법률(지역화폐법) — 지역화폐는 해당 지자체 내 사용만 허용. 광역 연계나 외부 플랫폼의 직접 발행은 지자체 조례와 행안부 승인 필요.
2. **전자금융 병목:** 선불전자지급수단 발행을 위해서는 금융당국(금융위원회) 등록 또는 핀테크 제휴사 확보 필수. 외국인에 대한 KYC 의무(전금법 시행령, AML 규정)를 자체 충족해야 함.

### 현실적 노선 평가: '제휴형 선불전자지급수단 + 간접 지역 가맹점 연동'

Land-In이 Phase 1~2에서 현실적으로 취해야 할 경로는 **독자 선불전자지급수단 발행보다 기존 선불전자지급수단(Trip.PASS 신한카드 또는 와우패스 등)과의 API 제휴 구조**가 맞다. 직접 지역화폐 발행이 아닌, 가맹점 간접 연동(가맹점이 지역화폐 + Land-In 두 가지를 수용)으로 시작하고, Phase 3 이후 금융당국 등록을 통한 직접 발행으로 단계적 전환하는 로드맵이 타당하다.

## 6. 확인된 사실 vs 추정

### 사실로 확인된 내용

- 비짓부산패스: 2024년 말 누적 31만 매 판매, 217억 원 매출. 부산관광공사 위탁 운영. 로카모빌리티(Mybi) 교통카드 연동. 가맹점 211개소(2025년 4월 기준).[^8][^10]
- 비짓부산패스: 외국인 전용, 외국인등록증·국내 통신사 인증 불필요[^1]
- 비짓부산패스: 모바일 패스는 대중교통 사용 불가, 실물카드도 별도 현금 충전 필요[^9]
- 탐나는전: 제주도 발행 지역화폐. 2026년부터 비즈플레이–제주은행 컨소시엄 운영대행. 운영 수수료 발행액 0.1%.[^17][^18]
- 탐나는전: 외국인등록증 보유자만 가입 가능. 한국 통신사 본인인증 필요[^2]
- 나우다: 제주관광공사 + 네이버파이낸셜 MOU. 2025년 8월 출시, 10월 기준 가입자 4만 명 돌파. 내국인 만 14세 이상만.[^19][^14]
- Trip.PASS: 로드시스템(eKYC) + 신한카드(금융) 공동 플랫폼. 2025년 3월 공식 출시. 여권만으로 가입 가능. 최대 200만 원 선불충전.[^29][^3]
- Trip.PASS: 신한카드 선불전자지급수단 발행. BC카드 343만 가맹점에서 신한카드 인프라로 전환.[^33]

### 합리적 추정 (공개 자료 부족)

- 비짓부산패스 무료 가맹점 이용 비용 정산 방식 (이용횟수 기반 vs 정액 협약) — 공개 자료 없음
- 비짓부산패스 민간 운영대행사 수익 구조, 수수료율 — 사업공고 금액(약 5.2억 원)만 확인
- 나우다 NFT의 구체 블록체인 체인(이더리움, Polygon 등) — 공개 미확인
- Trip.PASS의 이스타항공 기내 판매 실적, 현재 다운로드 수 정확한 수치 (10만+ 수준만 확인)[^30]
- Trip.PASS의 실제 거래량·활성 사용자 수 — 공개 자료 없음
- 탐나는전의 외국인 관광객 실제 사용 비중 (탐나는전 전체 사용액의 7.2%가 도외 가입자 기준이나 외국인 단독 데이터 없음)[^35]

## 7. 추가 조사 필요 항목

1. **비짓부산패스 무료가맹점 정산 단가:** 관광시설별 무료입장 제공 비용을 부산관광공사가 어떻게 보전하는지 — 정액 협약인지, 이용건당 정산인지 확인 필요. 조달청 계약서 또는 부산관광공사 감사 자료 확인 권고.
2. **Trip.PASS 실제 월간 활성 사용자(MAU) 및 가입자 수:** 로드시스템 또는 신한카드 투자설명회 자료, 앱 분석 서비스(앱애니, 모바일인덱스) 데이터 필요.
3. **나우다 블록체인 기술 구조:** 어떤 체인에서 NFT를 발행하는지, 외부 거래·양도 가능 여부. 제주관광공사 공식 기술 문서 필요.
4. **탐나는전 2026년 나우다-외국인 확장 계획 구체화 여부:** 2026년 외국인 여행 플랫폼 활용 계획이 실제로 추진되는지, 어떤 eKYC 솔루션을 사용하는지 확인 필요.
5. **부산페이(외국인 전용 지역화폐)와 비짓부산패스 연동 현황:** 부산페이 앱에서 비짓부산패스 모바일 패스 구매 연동 실제 이용률[^10]
6. **Trip.PASS의 항공사 파트너 확대 계획:** 이스타항공 단독에서 여타 LCC(제주항공, 진에어 등)로 확장 여부 — Land-In 항공사 파트너 전략과 직접 충돌 여부 판단에 필요.
7. **지역화폐법 개정 동향:** 행안부의 지역화폐 외국인 사용 허용 가능성. 2026년 지역사랑상품권법 개정 논의 여부.
8. **비짓부산패스 vs 부산 EMV 교통 단말기 도입 일정:** 2027년 도입 계획이 구체화되면 비짓부산패스의 교통카드 의존도 및 Land-In 교통 연계 전략에 영향.
9. **나우다의 제주 이외 지역 확장 가능성:** 제주관광공사가 타 지역 관광공사와 NFT 관광증 공동 운영 프레임 제안 여부.
10. **Trip.PASS의 선불전자지급수단 이용약관 및 환불 정책 전문:** 외국인 관광객이 잔액 환불 받을 때의 절차와 수수료 구조 — Land-In의 정산 설계에서 참조 필요.
